# Виджет редактирования пользовательских конфигурационных переменных (Qt/C++)

Проект представляет собой универсальный графический компонент (виджет) на базе фреймворка **Qt (C++)**, предназначенный для просмотра, создания, удаления и редактирования пользовательских переменных в табличном виде. 

Компонент разработан в рамках производственной практики и спроектирован с учетом паттерна разделения бизнес-логики, интерфейса и модели данных.

---

## 🚀 Основной функционал

* **Табличное представление:** Отображение данных в 3 колонки: `Ключ`, `Значение` и `Комментарий`.
* **Persistent Storage (Персистентность):** Автоматическое сохранение всех изменений в `QSettings` в формате **JSON** при любом редактировании.
* **Статические методы доступа:** Реализованы статические методы для быстрого чтения значений по ключу и получения списка всех доступных ключей из любой точки приложения без создания экземпляра виджета.
* **Валидация «на лету»:** Интерактивная проверка корректности ввода ключа (запрет пустых строк, дубликатов и спецсимволов — разрешены только латиница, цифры и нижнее подчеркивание `_`). При ошибке ввода ячейка подсвечивается красным цветом, и выводится `QToolTip` с подсказкой.
* **UX/UI:**
    * **Мягкий поиск:** Фильтрация строк в реальном времени с помощью `QSortFilterProxyModel` и визуальное подсвечивание (Highlighter) найденного текста в ячейках.
    * **Undo/Redo стек:** Полная поддержка отмены и повтора действий (`QUndoStack`) для операций добавления, удаления и редактирования ячеек (включая горячие клавиши `Ctrl+Z` / `Ctrl+Shift+Z`).
    * **Сохранение геометрии:** Виджет автоматически запоминает свои размеры и положение на экране при закрытии приложения.

## 🛠 Архитектура и технологии

При разработке были использованы следующие компоненты Qt:
* **`QAbstractTableModel`** — кастомная модель данных для эффективного управления строками таблицы (`VariableTableModel`).
* **`QStyledItemDelegate`** — кастомный делегат ячеек (`ValidatingDelegate`) для реализации валидации в момент ввода и кастомной отрисовки (`paint`) текста с подсветкой результатов поиска.
* **`QSortFilterProxyModel`** — прокси-модель для реализации быстрой сортировки и фильтрации без изменения исходных данных.
* **`QUndoCommand` & `QUndoStack`** — паттерн "Команда" для реализации истории действий.
* **`QJsonDocument`, `QJsonObject`, `QJsonArray`** — сериализация данных таблицы в JSON-строку для компактного хранения в `QSettings`.

## 📁 Структура исходного кода

* `GlobalVariant.h` / `.cpp` — Интерфейс `IGlobalVariant` и класс `GlobalVariant`, отвечающие за хранение, валидацию ключей и работу с `QSettings`.
* `VariableModel.h` / `.cpp` — Реализация табличной модели `VariableTableModel`, делегата `ValidatingDelegate` и классов Undo/Redo команд.
* `Vid11.h` / `.cpp` — Главный класс графического виджета, собирающий UI компоненты, настраивающий связи (signals/slots) и горячие клавиши.
